Job Summary

The Hands-On Tech Lead is an elite full-stack "Unicorn" Generalist who leads technical execution from the front by writing code every single day. In this role, you will sidestep endless theoretical debates and take complete ownership of end-to-end features, building high-performance TypeScript APIs, custom state-machine-driven backend systems, and the user-facing web experiences that connect them. This position requires an expert-level engineer with a strong bias to action who uses next-gen AI coding tools with extreme discipline as a mature utility, focusing heavily on AI engineering maturity to mathematically measure and prove the output reliability of custom agentic systems.

Responsibilities

  • Lead Technical Execution: Guide the development of end-to-end features for a lean team, making pragmatic trade-offs between rapid shipping and long-term code health.

  • Ship from the Front: Remain heavily hands-on, writing clean, production-grade code across the entire full-stack TypeScript ecosystem.

  • Architect AI & Agentic Systems: Move past basic implementations to deploy custom agentic workflows, state machine logic, and custom RAG pipelines.

  • Engineer Trust & Reliability: Build custom guardrails, traceability frameworks, and confidence/quality algorithms to mathematically measure, test, and prove LLM output reliability.

  • Execute with Ambiguity: Turn high-level, ambiguous ideas into functional, working code rapidly, prioritizing immediate execution over lengthy architecture meetings.

  • Set Engineering Standards: Establish rigorous code review processes, testing discipline, and guidelines for the responsible and mature use of AI coding assistants within the team.

Qualifications and Job Requirements

Must-Have Technical Expectations & Core Skills:

  • Language Expertise: Expert-level TypeScript proficiency with deep comfort utilizing advanced patterns (e.g., Generics and Discriminated Unions) to maximize shipping speed.

  • Backend Foundations: Deep Node.js experience, with a strong preference for Mastra or a proven background in building custom state-machine-driven backend logic.

  • Frontend Frameworks: Strong capabilities in modern web development; our primary stack is SvelteKit, but React or Vue wizards with the engineering DNA to pick up new frameworks fast are highly welcome.

  • Production AI Experience: A track record of deploying practical LLM-powered features or agents, paired with a solid understanding of token costs, evaluations, and prompt safety.

  • Pragmatic AI Tooling: Advanced, daily use of next-gen engineering assistants (e.g., Claude Code, Cursor) to automate boilerplate, combined with the strict engineering discipline to manually debug syntax walls in minutes.

  • Elite Full-Stack Generalist: The ability to jump across the entire web stack autonomously without being blocked by business-logic friction or basic syntax.

  • Remote Excellence: Outstanding English communication skills with comfort working in an async-first, highly autonomous environment.
