Senior Cloud Architect — Python & Apache Airflow (GCP Architect — Metro) via Topcoder.

We are seeking an experienced Senior Cloud Architect to design, implement, and optimize cloud-native solutions on Google Cloud Platform (GCP). This role combines deep expertise in Python development with advanced cloud infrastructure skills, focusing on migrating and managing complex workflows in a modern, scalable environment.

Responsibilities

  • Design and architect cloud-native solutions on GCP, leveraging Kubernetes for container orchestration and scalability
  • Develop and optimize Python applications and data pipelines using Apache Airflow for workflow automation
  • Implement Infrastructure as Code (IaC) using Terraform to provision and manage GCP resources
  • Configure and maintain observability solutions using Datadog for monitoring, logging, and alerting
  • Lead cloud migration initiatives, translating on-premises or legacy systems to GCP infrastructure
  • Collaborate with cross-functional teams to define cloud architecture standards and best practices
  • Optimize cloud costs and performance through resource management and architectural improvements
  • Maintain version control discipline using Git and establish CI/CD pipelines for deployment automation

Requirements

  • 5+ years as a cloud architect or senior backend engineer
  • Expert-level Python development and data engineering
  • Mandatory expertise in Apache Airflow for workflow orchestration
  • Advanced knowledge of GCP services and architecture patterns
  • Hands-on Kubernetes for container orchestration
  • Proficiency with Terraform (IaC)
  • Strong Datadog experience for observability
  • Demonstrated Git and collaborative development expertise

Nice to have

  • AWS, Azure, or multi-cloud experience
  • Prefect, Dagster, or Dask
  • BigQuery or Snowflake
  • Docker Compose / Swarm
  • GCP certifications (Cloud Architect, Cloud Engineer)

Duration: 10 weeks · Timezone: Any · Location: Remote · Workload: Full Time · Compensation: Negotiable · Start: Immediate
