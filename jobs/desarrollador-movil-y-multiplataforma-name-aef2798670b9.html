📱 Desarrollador Móvil y Multiplataforma | Proyecto Internacional | Modalidad Remota

En ReKluti buscamos Desarrolladores Móviles y Multiplataforma para integrarse a un importante proyecto internacional en Costa Rica, enfocado en el desarrollo, mantenimiento y evolución de aplicaciones móviles empresariales para plataformas iOS y Android.

Si te apasiona el desarrollo móvil y deseas participar en un proyecto de gran impacto con tecnologías de vanguardia, queremos conocerte.

📍 Condiciones de la oportunidad

  • Proyecto para un cliente internacional en Costa Rica.

  • Modalidad 100% remota.

  • Contratación bajo la modalidad de Staff Augmentation (asignación de profesionales por demanda de horas).

  • La asignación de horas dependerá de las necesidades del proyecto y del cliente final.

  • Proyecto con una duración estimada de hasta 4 años.

  • Inicio sujeto a la aprobación del proceso de selección y del cliente.

  • Compensación negociable, acorde con la experiencia y cumplimiento del perfil.

  • Disponibilidad para viajar a Costa Rica en caso de ser requerido por el proyecto.

💼 Principales responsabilidades

  • Desarrollar aplicaciones móviles nativas y multiplataforma.

  • Diseñar e implementar nuevas funcionalidades.

  • Integrar aplicaciones con servicios en la nube y APIs REST.

  • Gestionar tareas y seguimiento mediante Azure DevOps.

  • Elaborar documentación técnica y garantizar la calidad de las soluciones desarrolladas.

🎯 Perfil requerido

  • Bachiller universitario en Ingeniería en Sistemas, Ciencias de la Computación o carreras afines.

  • Mínimo 3 años de experiencia comprobable en desarrollo de aplicaciones móviles.

  • Haber participado en al menos un proyecto de software implementado en producción.

Tecnologías requeridas

  • .NET MAUI

  • MVVM

  • Swift

  • Objective-C

  • Xcode

  • Android Studio

  • React Native

  • Bootstrap

  • Responsive Design

  • REST API

  • JSON

  • SQL Server

  • T-SQL

  • Azure Services

  • Azure DevOps

  • Arquitectura SOLID

  • Test Driven Development (TDD)

📜 Certificaciones requeridas

Es indispensable contar con certificaciones vigentes y verificables en al menos las siguientes áreas:

  • Scrum Foundations o superior.

  • DevOps Foundation o equivalente.

  • Microsoft Azure (AZ-900, AZ-204 o superior).

⭐ Competencias

  • Pensamiento analítico.

  • Comunicación efectiva.

  • Trabajo en equipo.

  • Orientación a resultados.

  • Organización.

  • Calidad en el desarrollo de software.

  • Adaptabilidad a metodologías ágiles.

¿Cumples con el perfil? Postula y participa en un proyecto internacional con tecnologías de última generación.
